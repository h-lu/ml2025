项目三：用户分群分析与业务策略制定

项目背景

在商业分析中，用户分群（Customer Segmentation）是一种将用户基于其特征和行为模式划分为不同群体的方法。通过聚类分析，我们可以发现用户的自然分组，从而制定差异化的营销策略和服务方案。

项目目标

  1. 应用 K-Means 和 DBSCAN 聚类算法进行用户分群
  2. 评估和优化聚类效果
  3. 对不同用户群体进行特征分析
  4. 提出基于数据的商业策略建议

项目阶段

第一阶段：K-Means 聚类（第七周）

  1. 数据预处理
    • 特征选择与工程
    • 处理缺失值和异常值
    • 特征标准化（使用 StandardScaler）
  2. K 值选择
    • 使用肘部法则（Elbow Method）
    • 计算轮廓系数（Silhouette Score）
    • 结合业务理解确定最终 K 值
  3. 初步聚类
    • 使用选定的 K 值进行 K-Means 聚类
    • 使用 PCA 或 t-SNE 进行可视化
    • 初步评估聚类效果

第二阶段：优化与深入分析（第八周）

  1. DBSCAN 尝试（可选）
    • 使用 K-距离图选择 eps 和 min_samples
    • 运行 DBSCAN 并评估结果
    • 与 K-Means 结果对比
  2. 最终模型选择
    • 对比 K-Means 和 DBSCAN 结果
    • 基于评估指标选择最终模型
    • 确定最终的聚类方案
  3. 深入业务解读（核心）
    • 分析各簇的特征分布
    • 计算关键统计指标
    • 可视化簇间差异
    • 创建用户画像
    • 提出商业建议
